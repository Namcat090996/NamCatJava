Stanford - Dạy kinh nghiệm lập trình

Khóa học Java web for Developer tại Stanford cung cấp cho các bạn các mảng kiến thức quan trọng sau:

- Lập trình web cơ bản với HTML, CSS, Javascript và jQuery

- Lập trình với Servlet và JSP

- Lập trình với MySQL bằng JDBC

- Làm việc với hàm, tập lệnh truy vấn nâng cao trong MySQL

- Làm việc với Spring MVC từ cơ bản đến nâng cao

- Làm việc với các framework jQuery, Bootstrap, Hibernate, Spring MVC, Restful API,..

- Làm việc với Spring boot, JPA, Thymeleaf,...

- Project cuối khóa
